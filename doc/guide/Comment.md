# 评论区
## 评论礼仪
| |
|-|
|评论礼仪相关|
|●    禁止讨论代抢，脚本，虚假信息等|
|倡议内容|
|以下评论，让本站更温暖、更有爱。|
|●    与相关内容相关的评论|
|●    符合相关内容气氛的评论|
|●    专业的科普解说|
|●    请让你的评论“ 阳光、 理性、  平和、 友善|
|禁止内容|
|违反以下规则都将受到相应的处理。|
|●    违反法律法规：|
|发布违反国家相关法律法规的信息。包括但不限于发送色情、赌博诈骗、不实信息、非法网站等信息。|
|例：「*赌场*.com」、「成人电影*」|
|●    垃圾广告|
|禁止以曝光自己为目的，发送垃圾信息。包括但不限于发送大量链接、网站名、自己ID、qq群号等。|
|例：「欢迎来我qq群2323*」、「*.com欢迎你」|
|●    人身攻击|
|禁止一切针对其他用户的谩骂、羞辱、嘲讽。若对方有错在先，请善用举报、屏蔽等功能，切勿因在评论中互相谩骂而影响他人观看。|
|例：「前面的是s*么」、「你*飞了」|
|●    引战|
|1. 针对特定群体（包括但不限于：种族、国籍、宗教信仰、政治立场、身份职业、出身地区、兴趣爱好等），发布挑衅性质、攻击性质（包括但不限于明示与暗示的歧视、嘲讽、贬低、挑衅、辱骂等）的内容，或存在煽动他人进行上述行为的情况。|
|例：瘟疫给力点让这个垃圾种族快点灭族吧。|
|中国妹子？这群人怕已经是鬼子了吧国籍。|
|2. 对内容创作者（包括UP主、原作者等）及其作品发表恶意评论（包括但不限于明示与暗示的嘲讽、贬低、侮辱、挑衅、歧视等）的内容。|
|例：这是我见过最差劲的解说，UP主你可以滚了。|
|●    恶意刷屏|
|禁止单一用户恶意在短时间内发送过多相同评论。值得一提的是，在相关内容气氛烘托下，多用户发送的相同评论，属于共同的情感表达，不属于恶意刷屏。|
|例：「挡住老婆不给你们看」*10（以上均为同一个人发）|
|●    私人信息|
|评论是公众场所而非私人场所，请不要发送任何个人信息。包括但不限于表白个人、留下自己姓名、留下自己和他人联系方式等。|
|例：「李*到此一游~」「我手机号188****」|
|●    无关无意义、与相关内容不相关内容|
|评论中不应存在与相关内容完全无关或者完全无意义的内容。|
|例：「飘~」、「*倍速」、「你暴露了」「说暴露的你也暴露了」|
|●    低俗词汇|
|尽量少使用低俗词汇，尤其是当与原相关内容的氛围格格不入时。以上情况包括但不限于低俗空耳、低俗词汇等。|
|●    过度跟风|
|不鼓励过度刷流行词，尤其是与相关内容没那么相关的时候。当大家刷一个词时，往往预示着这个词快过气了。|
|例：「我可能喝了假x」（大量且持续时间长）「xxx我们走」（大量且持续时间长）|
|●    无关相关内容中刷口号|
|类似「无限大……」、「你指尖跳动……」等评论之所以令人感动，是因为他们发送在合适的时间、场所。但是，我们不鼓励在无关相关内容或不合适的时间过度刷口号。|
|例：「你指尖跳动……」（在非炮姐相关内容中全程刷）|
|●    误导他人|
|在评论里，一点点的错误都可能导致误导他人，或招致他人的反驳和攻击，因此请三思后发言。|
|例：「爱因斯坦发明了原子弹」|
|●    不懂得阅读气氛（KY）|
|请注意气氛，并看场合说话。我们鼓励发与相关内容氛围相称的评论，而非秀优越。|
|例：「只有我一个人觉得她不好看么？」「你们泪点真低|
