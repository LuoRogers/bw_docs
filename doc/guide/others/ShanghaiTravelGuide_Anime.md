# 上海 BW/BML 空闲时间的可去之处

## **黄浦区**

### **// 购物**

**1、百联ZX创趣场**
>一个专卖周边的商场，价格适中，适合逛和社交。
- **营业时间：** 早10点～晚10点
- **地址：** 中国上海市黄浦区南京东路340-372号
- **推荐交通：** 地铁2号线、地铁10号线 - 南京东路站（**1号口**步行70米）

**2、百米香榭商业街**
>一些周边小店铺聚集的地方，价格不一，建议货比货。
- **营业时间：** 早10点～晚10点（具体看店铺营业时间）
- **地址：** 中国上海市黄浦区广西北路252号中福城二期内
- **推荐交通：** 地铁1号线、地铁2号线、地铁8号线 - 人民广场站（**14号口**步行220米）

**3、上海迪美购物中心**
>一个地下商场，有一些小店铺，适合逛街和比价。
- **营业时间：** 早10点～晚9点
- **地址：** 中国上海市黄浦区武胜路218号
- **推荐交通：** 地铁1号线、地铁2号线、地铁8号线 人民广场站（**1号口**）

### **// 逛街**

**1、外滩/北外滩**
- **外滩：** 可以看到浦西的万国建筑和浦东的高楼大厦，适合第一次来上海的游客。
- **北外滩：** 位于虹口区北边，人比外滩少一些，适合安静散步和拍照。
- **推荐交通：**
- 外滩：地铁2号线、地铁10号线 南京东路站
- 北外滩：地铁12号线 国际客运中心站

**2、苏州河**
- 适合悠闲的散步，感受城市的氛围。

### **// 博物馆**

- 上海博物馆、上海城市规划展览馆、一大会址、世博会博物馆

## **静安区**

### **// 购物**

**上海静安大悦城**
>内有动漫主题店和潮玩星球等，定期有联动快闪店和咖啡厅。
- **营业时间：** 早10点～晚10点
- **地址：** 中国上海市静安区西藏北路166号
- **推荐交通：** 地铁12号线、地铁8号线 曲阜路站（1号口）

### **// 逛街/博物馆**

- 苏州河、四行仓库、邮政博物馆（暂停开放）

## **徐汇区**

### **购物**

1. **美罗城**
- 内有Jump shop和良笑社等店铺，适合动漫迷。
- **营业时间：** 早10点～晚10点
- **地址：** 中国上海市徐汇区肇嘉浜路1111号
- **推荐交通：** 地铁1号线、地铁11号线、地铁9号线 徐家汇站（10号口）

### **逛街/博物馆**

- 武康路历史文化名街、徐汇滨江绿地、上海公安博物馆

### **特别介绍**

- **米哈游总部**
- **地址：** 中国上海市徐汇区苍梧路519号光启园四期1号楼
- **开放时间：** 周六周日及法定假日10:00～12:00、14:00～18:00
- **交通：** 公交120路、205路 苍梧路钦江路站；地铁9号线、15号线 桂林路站（10号口步行约700米）

## **杨浦区**

### **逛街/购物**

1. **五角场中心广场（下沉广场）**
- 地下通道连接中心，附近有多个商场，适合消磨时间。
- **推荐交通：** 地铁10号线 五角场站（5号口）

2. **大学路**
- 小步行街，有多家店铺，包括动漫相关的店。
- **推荐交通：** 地铁10号线 江湾体育场站（9号口）

3. **B站总部（国正中心）**
- 适合拍照打卡，无实际参观项目。
- **地址：** 中国上海市杨浦区政立路485号
- **推荐交通：** 地铁10号线 江湾体育场站（10号口步行约900米）

## **浦东新区**

**逛街/观景**

1. **陆家嘴金融城**
- 适合拍照和观光，有多处高楼大厦景观。
- **推荐交通：** 地铁2号线 陆家嘴站
